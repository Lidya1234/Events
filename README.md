# User details
A simple API for add , getting ,deleting and updating Events


## Built With

- Nodejs 
- Expressjs
- Mongodb-memory-server
- Typescript



## Getting Started

To get a local copy up and make it run, follow these simple example steps.
- cd ./API/
- npm install
- npm start



### Prerequisites

- code editor
- Browser
- Node.js installed



### Setup

- install code editor
- Browser
- Postman
